# Juego del Número Secreto
Un juego desarrollado con **JavaScript**, **HTML** y **CSS**, donde el jugador debe adivinar el número.

## Funcionalidades
- Genera un número aleatorio entre **1 y 100**.
- Indica si el número secreto es **mayor** o **menor** que el ingresado.
- Cuenta la cantidad de intentos realizados.
- Evita repetir números secretos ya utilizados hasta agotar todas las posibilidades.
- Permite reiniciar la partida con un nuevo número.

## Tecnologías
- HTML,
- CSS,
- JavaScript.

## Cómo ejecutar el proyecto
1. Clona este repositorio o descarga los archivos.
2. Abre el archivo `index.html` en tu navegador.
3. Ingresa un número entre **1** y **100** e intenta adivinar el número secreto.

## Estructura
- `index.html` - Estructura de la página.
- `style.css` - Estilos de la aplicación.
- `app.js` - Lógica del juego.

## Lo aprendido
Este proyecto pone en práctica conceptos fundamentales de JavaScript como:
- Funciones,
- Condicionales,
- Arreglos (`Array`),
- Manipulación del DOM,
- Generación de números aleatorios,
